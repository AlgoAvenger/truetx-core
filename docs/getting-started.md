# Getting Started

## 1. Clone the repository

```bash
git clone https://github.com/your-username/truetx-core.git
cd truetx-core
```

## 2. Setup Environment

Copy the `.env.example` to `.env` and fill in the required fields:

```bash
cp .env.example .env
```

## 3. Run Scripts

```bash
python push_meta.py
```

> This will set up your repo metadata including CI, templates, etc.
